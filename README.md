# hypernova-php-react-poc
Proof of concept showcasing react micro-frontend integration in existing PHP app using Airbnb hypernova

# How to run
Hypernova app
`` npm install ``
`` npm start ``

PHP app
`` composer install `` then run ``index.php`` in any php server
